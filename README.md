# ByteForms

### Installation

1. Install Node.js

2. Install pnpm
   [link](https://pnpm.io/installation)

3. Install dependencies

```bash
pnpm install
```

4. Configure .env
   Create a .env files that contains same variables as .env.example

5. Run development environment

```bash
pnpm run dev
```

### Production

1. Build

```bash
pnpm run build
```

2. Run

```bash
pnpm run start
```
